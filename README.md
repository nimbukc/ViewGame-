# ViewGame- Soccer Stats Dashboard 
ViewGame gives you a simple to use and easy to read soccer stats dashboard with the ability to search by league or team, get current league standings and the current results of matches. You can also add your favourite teams to the dashboard for easier tracking.

## Description 
Many soccer fans have difficulty finding up to date statistics for their favourite teams in a coherent manner. ViewGame helps you find that withan easy to use dashboard providing real-time Bundesliga (Germany's top league) information powered by OpenLigaDB.

## Target Browers 
ViewGame is designed for desktop browsers including:
1. Google Chrome
2. Safari 
3. Mozilla Firefox
4. Microsoft Edge
